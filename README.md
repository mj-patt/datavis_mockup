# datavis_mockup

How to view the vis
1. Open 2 different browser windows
2. Load the page of the left panel under <localhost>/screen_left.html
3. Load the page of the right panel under <localhost>/screen_right.html

structure:
- default state shows the overview of authors NL diagram
- layer 0 : papers NL
- layer 1 : authors NL
- layer 2 : institutions NL
- layer 3 : countries NL
- layer 4 : continents NL

How to change states:
- from overview to folded : double tap/click anywhere on the left panel
- from folded to overview : double tap/click anywhere on the top part of the left panel
